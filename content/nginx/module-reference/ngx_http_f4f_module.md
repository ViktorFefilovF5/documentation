## `f4f`

**Contexts:** `location`

Turns on module processing in the surrounding location.

## `f4f_buffer_size`

**Syntax:** *`size`*

**Default:** 512k

**Contexts:** `http, server, location`

Sets the *`size`* of the buffer used for
reading the `.f4x` index file.
